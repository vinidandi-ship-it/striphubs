# StripHubs

Production-ready React + TypeScript + Vite web app for a live cam directory with Stripchat affiliate monetization.

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM

## Local Development
Prerequisites:
- Node.js 20+
- npm 10+

Install dependencies:
```bash
npm install
```

Start dev server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Preview production build:
```bash
npm run preview
```

## Routes
- `/`
- `/live`
- `/cam/:category`
- `/tag/:tag`
- `/model/:name`
- `/search`
- `/privacy`
- `/terms`
- `/cookies`

## SEO
- Dynamic page title and description
- Canonical tags
- JSON-LD `WebSite` and `BreadcrumbList`
- `robots.txt`
- `sitemap.xml` generated by `npm run generate:sitemap`

## Affiliate Integration
Affiliate ID used globally:
`d28a8a923e19b6fd3ed0c160238cdfed71b13f759191c9457b28797b81780881`

Model outbound links follow:
`https://stripchat.com/{modelname}?userId=<AFFILIATE_ID>`

Widget embed is injected dynamically via `StripchatWidget` component.

## Vercel Deploy
1. Push repository to GitHub.
2. Import project in Vercel.
3. Framework preset: `Vite`.
4. Build command: `npm run build`.
5. Output directory: `dist`.
6. Optional env: `VITE_SITE_URL=https://your-domain.com`.
7. Deploy.

SPA routing is handled through `vercel.json` rewrites.

## Deploy Checklist
- [ ] `npm install` completed successfully
- [ ] `npm run build` passes locally
- [ ] `public/sitemap.xml` generated/updated
- [ ] `public/robots.txt` points to production sitemap URL
- [ ] `VITE_SITE_URL` set in Vercel project settings
- [ ] Domain connected and HTTPS active
- [ ] Age gate and cookie consent visible on first visit
- [ ] Core pages load correctly on mobile and desktop

## Notes

- This project is intended for adults only (18+).
